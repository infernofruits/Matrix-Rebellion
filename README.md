# ▲ Matrix Rebellion: Holocron Override
**Matrix Rebellion** is an action-packed 2D space hack-and-slash roguelike created using nothing but one file that utilizes HTML5 Canvas and JavaScript. Created in the spirit of Star Wars and cyberpunk genres, you control a rebel infiltration spacecraft that flies through the Empire's encrypted HoloNet Grid.


## 🌌 The Lore

The Empire has encrypted all the history of the galaxy into the huge **HoloNet Grid**. You are a rogue hacker-pilot and your job is to connect to this network and breach the encrypted data of Imperial Security Nodes to help ignite the rebellion.

Mission: Infiltrate the Imperial Data Core, evade or destroy automated security sentries and collect glowing **Kyber Shards** that will allow you to unlock classified **Jedi Holocrons**. Each time you override a security node, you get an opportunity to change your system's code by adding illegal mod patches. 

The Empire is aware of your presence in their system. **How much can you withstand the override?**

## 🚀 Features

* **Fluid Vector Combat:** Slip past enemy droids by dodging and shooting using physics-based motion tracking (`Math.sin`/`Math.cos`).
* **Magnetic Kyber Shards:** Kill off the enemy droids to collect the colorful data shard pieces. Expand your magnet field to attract the data pieces from afar.
* **Modular Upgrades:** Level up every time you get a new shard and then pause the main frame to pick three random hardware mods (i.e., *Split-Stream Compiler*, *Overclock Overdrive*, *Nano-Repair Patches*).
* **Scalability:** The game grid will scale up according to your threat level and spawn aggressive yellow *Chasers* and huge purple *Core Heavy Bosses*.
* **Zero Dependencies:** Written in vanilla ES6 JavaScript & CSS. No engines/frameworks needed.


🎮 How to Play

### Controls
| W / A / S / D (or Arrows) | Acceleration / Move on the grid |
| Mouse Pointer | Targeting Reticle |
| Left Mouse Button | Shoot synchronized pulse lasers |

### Upgrades Modules
* ⚡ **OVERCLOCK OVERDRIVE:** Increases Speed by 15%.
* ⚙️ **CYCLIC TRIGGER:** Reduces Reload Time by 18%.
* 💥 **OVERCHARGED CORE:** Increases Laser Damage by 25%.
* 🧲 **DATA SYNC HORIZON:** Increases Shard Magnet Effect by 40%.
* 🛰️ **SPLIT-STREAM COMPILER:** Creates one more projectile (+1 up to max 3).
* 🔧 **NANO-REPAIR PATCH:** Restores 40% of current HP.
